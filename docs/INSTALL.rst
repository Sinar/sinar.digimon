sinar.digimon Installation
--------------------------

To install sinar.digimon using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``sinar.digimon`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        sinar.digimon

* Re-run buildout, e.g. with:

    $ ./bin/buildout
